# Bank_expense_tracker
A program that tracks my expenses every month. this program is made for ING bank in Germany 
 First we need the psd2 api from Ing bank. so we neet to go into https://developer.ing.com/openbanking/resources/get-started/psd2 and complete some steps to acquire some certificates and some access tokens. 
 Second we install openssl and we convert the .cer file to pem and the .key files to unencrypted key.
we need to create a developer account to ING for ING to provide us  the client id ,client secret
